# Multiprecision matrix exponential

This repository contains the code that produces the figures and tables in the
technical report:

M. Fasi and N. J. Higham, An arbitrary precision scaling and squaring algoirhtm
for the matrix exponential. Technical Report 2018.XX, Manchester Institute for
Mathematical Sciences, The University of Manchester, UK, Nov 2018.

## Dependencies

The code in this repository requires the Advanpix Multiprecision Computing
Toolbox for MATLAB (www.advanpix.com).
